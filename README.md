## About The Project

Im Modul 426 müssen wir Scrum lernen. Um dies direkt in einem Sprint umzusetzen haben wir dieses Projekt erstellt. Es ist eine Website / App eines Zoos um Tiere für Besucher zu katalogisieren und sonstige informationen darzustellen.

## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Cattlx/M426-ZooApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Run the project
   ```sh
   npm run dev
   ```
